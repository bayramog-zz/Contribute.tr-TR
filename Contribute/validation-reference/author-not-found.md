# <a name="author-not-found"></a>author-not-found

## <a name="warning"></a>Uyarı

`Invalid value for author: '{value}' is not a valid GitHub ID.`

## <a name="resolution"></a>Çözüm

`author` değeri olarak geçerli bir GitHub kimliği ekleyin.
